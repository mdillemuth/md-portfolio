Some of my projects.

myFightFlix Full-Stack App

This project is a full-stack single-page application built with the MERN stack. My role included designing and developing both the front-end and the back-end. The app uses RESTful architecture, Redux state management, and is used through a front-end built in React.

The app features client-side and server-side verification, an alert system implemented with Redux, sorting and filtering with lodash, and is fully responsive. Users can create an account and add movies to their list of favorites.

Chat App

A mobile chat app that provides users wiht a chat interface and options to share images and their location. Developed using React Native and Expo.

The app uses Google Firestore as its serverless document database. Gifted chat library is used in order to provide the chat interface.

Meet App

A serveless progressive-web app using the Google calendar API that users can install and use offline. Features authorization with AWS lamba serverless functions and OAuth2 authentication flow. The app is developed with a test-driven development approach aided by jest and enzyme libraries.

Users are able to use an auto-complete searching feature with suggestions, view data visualization implemented with recharts, and install the app on their home screen for offline use.

Pokédex App

This project was implemented in HTML, CSS, & JavaScript using the RESTful Pokédex API. Users are able to filter the list by searching for a specific pokémon. The app implements modals, contact form validation, a back-top-top button, and accordion components.

The app has consistent styling with fundamentally sound design features including a reasonable use of visual hierarchy, color & contrast, alignment, and whitespace.
